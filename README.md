# Algorand Project: Decentralized Voting System

## 📌 Description
This project is built on the Algorand blockchain. It aims to create a secure, transparent, and tamper-proof voting system where users can cast votes and verify results without relying on a central authority.

## 🚀 Features
- Secure voting using Algorand blockchain
- Transparent and immutable vote records
- Simple user interface for casting votes
- Real-time vote counting

## 🛠️ Tech Stack
- Algorand Blockchain
- Python (for backend and smart contracts)
- PyTeal (for writing smart contracts)
- JavaScript (optional frontend)

## 📂 Project Structure
- /contracts → Smart contracts written in PyTeal
- /frontend → User interface for voting
- /backend → Backend logic and blockchain interaction
